01 · what it checks

Three questions every production model needs answered.

🧠

Explainability

Auto-selects TreeExplainer, LinearExplainer, or KernelExplainer. Global importance and per-prediction breakdowns.

ShapAnalyzer
⚖️

Fairness

Demographic parity, equalized odds, and disparate impact against the EEOC 4/5ths rule — across any protected attribute.

FairnessEvaluator
📊

Drift

PSI and KS-test per feature, plus ADWIN and Page-Hinkley for concept drift on live prediction streams.

DriftDetector
02 · setup

Three lines from install to insight.

# install pip install -r requirements.txt # audit any trained model from mlens import ModelAuditor auditor = ModelAuditor(model, X_train, X_test, y_test, sensitive_features=df_test["gender"]) report = auditor.run() report.save("audit_report.html")
03 · interfaces

Use it however your team works.

  • CLI mlens audit model.pkl X_test.csv y_test.csv
  • Python API ModelAuditor(model, …).run()
  • Dashboard streamlit run dashboard/app.py
  • REST API POST /audit via FastAPI + Docker
  • MLflow / W&B log every audit as a tracked run
# start the live dashboard streamlit run dashboard/app.py # or run it as a service docker-compose -f docker/docker-compose.yml up # API docs at http://localhost:8000/docs
